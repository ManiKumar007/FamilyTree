## Why GPG Signing is Disabled

Your global git config uses work account credentials with GPG signing enabled. To prevent conflicts and ensure commits use your personal account without requiring GPG keys, GPG signing is disabled for this repository only.

## Verify Configuration

To check the current configuration:

```powershell
# Check local config (this repository only)
git config --local --list | Select-String -Pattern "user|gpg"

# Check global config (all repositories)
git config --global --list | Select-String -Pattern "user|gpg"
```

## If You Need to Reconfigure

# Disable GPG signing (local to this repo)
